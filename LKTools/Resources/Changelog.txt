V.0.4.0.0
# Features
 ## Allgemein:
- Änderung des Einstellungsfensters (Tab-Reihenfolge)
- Benutzer kann über ausgeblendete Objekte beim Start von Word informiert werden (Einstellung) --> funktioniert nur beim ersten Start von MS Word und nicht wenn mehrere Dokumente gleichzeitig geöffnet werden.
- Benachrichtigung bei manueller Update-Überprüfung, falls kein Update gefunden wurde
- Im Dialogfenster Einstellungen gibt es nun eine Schnellhilfe 

 ## Objekte:
- Grafische Darstellung der Notenverteilung 
- Hinzufügen einer Vorlage für Aufgaben (Anzahl der Zeilen ist über Ribbon einstellbar)
- Hinzufügen einer Vorlage für Prüfungsaufgaben (Anzahl der Zeilen ist über Ribbon einstellbar)
- Hinzufügen einer Vorlage für Leistungsnachweise (Anzahl der Zeilen ist über Ribbon einstellbar)
- Hinzufügen von MC (Multiple Choice) Aufgaben (Anzahl der Zeilen ist über Ribbon einstellbar)
- Hinzufügen von Richtig/Falsch Aufgaben (optional mit Begründung) (Anzahl der Zeilen ist über Ribbon einstellbar)
- Textfarbe der eingefügten Linearen Vorlage ist automatisch die Ausblendfarbe
- Der Style (gepunktet, gestrichelt, durchgezogen) und die Breite der linierten Vorlage kann in den Einstellungen festgelegt werden.
- Alle Buttons im Reiter Einfügen können einzeln ein- oder ausgeblendet werden.
- Auch Formen lassen sich nun automatisch in der Ausblendfarbe färben
- Wird ein Objekt gefärbt, dass mit der Eigenschaft "Text in Zeile" eingefügt ist, kann es bei Bedarf schnell umgewandelt werden.

 ## Ausgabe:
- PDF Ausgabe in Unterordner oder festen Pfad möglich
- Anzeige des Speicherdialogs, falls PDF Export gewünscht und Datei noch nicht gespeichert.
- Schnellauswahl der PDF Ausgabe (Lösung / Ohne Lösung) im Ribbon
- Schnellauswahl des Schnelldrucks (Lösung / Ohne Lösung) im Ribbon

# Bugfixes:
- Fehler in der Gültigkeitsprüfung der Notenbereiche im Einstellungsfenster
- Fehler in der Gültigkeitsprüfung der Notenwerte in der Notentabelle
- Korrektur der Überprüfung von ausgeblendeten Inhalten
- Der Status (ContentHidden) in den Dokumenteneigenschaften gibt an, ob Inhalte ausgeblendet sind. Das Symbol im Tab ist nicht aussagekräftig, wenn mehrere Dokumente verwendet werden.
- Überprüfung auf ausgeblendete Inhalte erfolgt erst nach vollständigen Laden des Dokuments
- Druckfunktionen basieren auf Dokumenteneigenschaft "Status" und nicht mehr auf Basis des Symbols.
- LKTools funktioniert nun auch korrekt, wenn ein Willkommensbildschirm beim öffnen von Word ausgewählt wird.
- Die Positionierung der Textbox "Lösungsvorschlag" wurde überarbeitet.
- Änderung des Button-Designs
- Verbesserung von Rechtschreibfehler
- Optimierung der Notenerstellung (Weißer Hintergrund, dickerer Rand, Schriftgröße 11, Textfarbe rot, Kein Abstand davor und danach, Zeilenabstand 1.0

# Hinweise:
- die Überprüfung, ob ausgeblendete Inhalte vorhanden sind, erfolgt nur, wenn ein Word Dokument geöffnet ist.

Installationshinweise LKTools V.0.4.0.4
1. Entpacken Sie die Datei <LKTools V.0.3.0.12.zip> in einen beliebigen Ordner.
2. Öffnen Sie den Ordner und klicken Sie auf <setup.exe>.
3. Die Funktionen sind im Handbuch <LKTools V.0.4.0.pdf> beschrieben.
4. Für Testzwecke können Sie das Word Dokument <LKTools Testdoc.docx> verwenden.

Install
1. Unpack the <LKTools V.0.4.0.0.zip> to any folder
2. Click on <setup.exe> to install LKTools.
3. See document Dokumentation <LKTools V.0.4.0.pdf> for further details (only German)
4. You can use the document LKTools <LKTools Testdoc.docx> for testing purposes.
